# 🍔 Foodix - Food Ordering Website 

## 📌 Overview
A modern food ordering website with interactive UI, allowing users to browse food items, view details, and simulate ordering experience.

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)

---

## ✨ Features
- 🍕 Food listing UI
- 📄 Product detail pages
- 🛒 Cart simulation
- 📱 Responsive design
- 🌙 Dark mode

---

## 🧠 Key Learnings
- UI/UX for food apps  
- Component reuse  
- Responsive design  

---
